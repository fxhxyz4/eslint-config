### @fxhxyz/eslint-config

#

### how to use?
+ ***npm***
```
npm install @fxhxyz/eslint-config -D
```

+ ***yarn***
```
yarn add @fxhxyz/eslint-config -D
```

+ ***pnpm***
```
pnpm add @fxhxyz/eslint-config -D
```

+ ***usage commonjs***
```js
// .eslintrc.cjs
module.exports = {
  extends: ["@fxhxyz/eslint-config/nodejs.js"]
};

// nodejs.js
// nodets.js
// webjs.js
// webts.js


// package.json
"scripts": {
    "lint": "eslint ."
}
```
